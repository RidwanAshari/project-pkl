# Sistem Manajemen Aset 🏢

Aplikasi buat ngelola barang-barang kantor/perusahaan biar gak pada ilang atau ketuker. Dari laptop, meja, mobil sampe gedung bisa di-track!

## Buat Siapa Nih?
- **Perusahaan** yang punya banyak aset
- **Sekolah/Kampus** yang punya inventaris banyak
- **Rumah Sakit** yang perlu track alat medis
- **Pemerintah** buat ngelola barang inventaris

## Bisa Ngapain Aja?

### 🎯 **Lacak Barang Gampang**
- Kasih QR Code di tiap barang, scan langsung tau detailnya
- Tau barangnya lagi di mana dan siapa yang pegang
- Catet kapan terakhir dipakai/diperbaiki

### 📱 **Fitur Kerennya**
- **QR Code Generator** - Bikin QR buat tiap barang
- **Scan & Go** - Pakai HP scan QR, langsung liat detail
- **History Tracking** - Tau barangnya udah pindah tangan berapa kali
- **Laporan Otomatis** - Export data ke PDF/Excel
- **Reminder** - Ingetin kapan perlu servis/periksa

### 🚗 **Khusus Kendaraan**
- Simpan data STNK, BPKB
- Catet riwayat servis
- Hitung biaya operasional
- Jadwal ganti oli & perawatan

## Teknologi yang Dipake
- **Laravel 10** - Framework PHP terbaru
- **MySQL** - Database yang stabil
- **Bootstrap 5** - Tampilan responsif
- **QR Code Library** - Generate QR kualitas tinggi
